# 🛡️ ABDULLAH RECON-X V0.1

RECON-X is a comprehensive information gathering and reconnaissance tool designed for security researchers and penetration testers. It automates several manual tasks required during the initial phase of a security audit.

---

## 🚀 Features

* **Reverse IP Lookup:** Identify other websites hosted on the same server/IP address.
* **SQL Injection Scanner (Basic):** Quickly check URL parameters for common SQL error signatures.
* **Hidden Files Finder:** Automatically detects `robots.txt` and `sitemap.xml` files.
* **CMS & Tech Detector:** Identifies CMS platforms like WordPress or Joomla and detects server headers.
* **IP Geo-Location:** Provides geographic details (Country, City, ISP) and a direct Google Maps link.
* **Subdomain Dictionary Attack:** Brute-forces common subdomains to map out a target's infrastructure.
* **WHOIS Lookup:** Retrieves domain registration and ownership information.
